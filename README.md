# SpringRestAPI

A REST API-driven Web Application using Spring Boot (Java) and testing with Serenity BDD.

## Getting Started

### Clone repository:
1. From your intended location, run: ```git clone git@github.com:acary/SpringRestAPI.git```
2. CD to the folder


## Run app:
1. Run:  ```./gradlew bootRun```

## Run tests:
1. Run: ```./gradlew test```
